## 📋 Installation & Quick Start

```bash
# Clone the repository
git clone https://github.com/your-username/omnipanel-core.git
cd omnipanel-core

# Install dependencies
npm install

# Build all packages
npm run build

# Start development server
npm run dev:web
```
